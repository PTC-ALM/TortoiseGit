TortoiseSI

Windows Explorer Extension for PTC Integrity

TortoiseSI is a free, open-source Microsoft® Windows® shell extension client for PTC Integrity.

PTC Integrity Lifecycle Manager is a cross-platform, enterprise-class software configuration management and versioning solution.

TortoiseSI exposes PTC Integrity views, source commands, and version-controlled member actions on Windows files and folders through Windows Explorer.

Version 1.0 of TortoiseSI has been released! Please see the user documentation for a description of the integration functionality.

Known issues and planned features that are not yet implemented can be found in the GitHub issue list.

TortoiseSI is developed as an Open Source project under the GNU General Public License (GPL-2.0).
